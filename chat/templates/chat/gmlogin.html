Welcome to TigerMeet
TigerMeet is a web application that enables Princeton University students to connect with other students for recreation, transportation, and academic coordination. It is divided into sections of general interest (i.e. Sports, Transportation) each with a general chat as well as the ability to create specific events with chatrooms for users to connect.

Login to TigerMeet with GroupMe

How It Works

Login using GroupMe
Check out six different categories: 1) Sports, 2) Working Out, 3) Video Games, 4) Transportation, 5) Problem Set Groups, 6) Miscellaneous.
Join and chat in the general group chat for each category
Check out student-made events under each category, each with their own event group chat
